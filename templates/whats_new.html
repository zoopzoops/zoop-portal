What's New

Latest updates and changes to Zoop Portal

v1.5.1 May 06, 2026 Latest ▼
  • NewBatch Approvals — admins can now approve requests in bulk.
  • FixedManual Approval Bug — admins can approve shows again.
v1.5 April 26, 2026 ▼
  • NewSeason Selection — when requesting a TV show, a modal lets you choose which seasons to download, or select New Episodes Only
  • NewManage Seasons — approved shows have a Manage button to add/remove seasons, toggle new episode monitoring, and trigger searches
  • NewSearch & Filter on Requests — filter by title, username, type or status by clicking the stat cards
  • NewSearch on Downloads — filter active downloads by name in real time
  • NewAlready Exists Check — submitting a request for something already in the library now shows a clear error
  • NewWhat's New page — you're looking at it!
  • ImprovedSeasons display — shown as a compact expandable pill in the requests table
  • FixedSpecial characters in titles — apostrophes and colons no longer break the Manage button
v1.4 April 24, 2026 ▼
  • NewPer-user Auto-Approve — admins can enable auto-approve per user with individual daily limits
  • SecurityLogin Attempt Locking — accounts disabled after 5 failed attempts, counter resets on success
  • NewFavicon — Zoop Portal now has a browser tab icon
v1.3 April 23, 2026 ▼
  • NewHigh Contrast Mode — toggle in the nav bar, preference saved per account
  • ImprovedDownloads page cleanup — seeding torrents and unnecessary stats removed
  • ImprovedDashboard cleanup — removed the Available stat counter
v1.2 April 23, 2026 ▼
  • NewLive Downloads page — progress bars, speed and ETA, auto-refreshes every 3 seconds
  • NewAdmin notifications — browser notifications when new users register or requests come in
  • SecuritySession kick — disabled accounts are immediately logged out
  • NewUser management — admins can disable, enable, reset passwords and delete accounts
v1.1 April 23, 2026 ▼
  • ImprovedAuto media type detection — no more selecting Movie or Show manually
  • SecurityPassword requirements — minimum 12 characters with uppercase, lowercase, number and symbol
  • ImprovedDuplicate detection — submitting the same IMDB link twice now shows a clear error
v1.0 April 23, 2026 Initial Release ▼
  • NewZoop Portal launched — media request system connected to Radarr and Sonarr
  • NewUser accounts — register and request access, admin approves before login
  • NewAdmin panel — approve/reject user registrations and media requests
  • NewPublicly accessible — hosted at portal.weeshnaw.com via Cloudflare Tunnel
